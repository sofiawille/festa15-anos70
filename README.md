# Sofia 15 Anos — Festa Groovy Anos 70

Site da festa de família dos 15 anos da Sofia, com tema **anos 70 / flower power**.

Festa para os amigos (anos 80): [festa15-anos80](https://github.com/sofiawille/festa15-anos80)

## Como rodar localmente

Pré-requisitos: Node.js

1. Instalar dependências: `npm install`
2. Rodar em desenvolvimento: `npm run dev`
3. Acessar http://localhost:3001

## Deploy

Push para `main` → GitHub Actions faz build e publica em GitHub Pages automaticamente.

URL: https://sofiawille.github.io/festa15-anos70/
